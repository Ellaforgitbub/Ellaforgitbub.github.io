    1. 我學過的標籤
      • 隱藏訊息標籤
        • <title>

          語意標籤，表示網頁的標題，通常在標籤<head>中

        • <meat>

          將這份網頁的資訊，提供給瀏覽器、社群網站或搜尋引擎，
          可提供的資訊包含了頁面編碼、作者資訊、關鍵字、頁面描述、網站縮圖、頁面發布時間…等。

        • <head>

          這個標籤內可以儲存只想給瀏覽器的資訊，不會顯示在網頁內，但是如果有文字或者其他有顯示功能的仍會被
          顯示於網頁內。

      • 效果標籤
        • <strong>

          表示這個字的重要性，屬於語言標籤之一，對SEO很有幫助

        • <em>

          語意標籤，與<i>效果相同，只是更加強調這個被標記元素的重要性

      • 圖片
        • <img>
          屬性(Attribute)
          • src

            載入圖片網址

          • alt

            如果圖片沒有出來，顯示文字代替圖片

          • width

            寬度(單位:相素)

          • height

            高度(單位:相素)

          • title

            滑鼠移到圖片上時自然浮現的說明

      • 輸入
        <input>的屬性(Attribute)
        type
        • date

          可以讓使用者選取日期

        • time

          可以讓使用者選取時間

        • range

          可以讓使用者選取範圍

        • password

          可以讓使用者輸入的資料隱藏

        placeholder

        輸入提示
